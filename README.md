Küresel Turizm – Turizm Rezervasyon Web Uygulaması

 Proje Amacı

Bu projenin amacı, Django framework kullanılarak geliştirilen bir turizm web uygulaması oluşturmaktır.
Uygulama sayesinde kullanıcılar:

- turistik destinasyonları görüntüleyebilir,
- kullanıcı hesabı oluşturabilir ve giriş yapabilir,
- destinasyonlar için rezervasyon yapabilir,
- kendi rezervasyonlarını görüntüleyip silebilir.

Ayrıca yönetici paneli aracılığıyla destinasyonlar, kullanıcılar ve rezervasyonlar yönetilebilmektedir.
